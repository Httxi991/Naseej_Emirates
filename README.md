# Naseej Emirates — Mobile-First Competition Prototype

A mobile-first static prototype designed for GitHub Pages.

## What changed
- Removed the fake phone frame, time, and battery bar.
- Removed the reset button.
- Moved Humaid AI to a smaller bottom floating action button.
- Rebuilt the UI to feel more like a real app, with a clean bottom navigation and modern card layout.
- Kept Digital ID / Emirates ID onboarding instead of manual name/age signup.
- Optimized for mobile screens first.

## Files
- `index.html`
- `styles.css`
- `script.js`

## Run locally
Open `index.html` in a browser, or upload the folder to GitHub Pages.
