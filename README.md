# Naseej Emirates — iOS Competition Prototype v2

Updated competition prototype for GitHub Pages.

## What changed
- Replaced manual sign-up with **Digital ID sign in**.
- Removed manual **name / age entry** from onboarding.
- Turned **Humaid AI** into a full **chat-style conversation screen**.
- Cleaned the UI so it feels more focused and less crowded for judging.

## Files
- `index.html`
- `styles.css`
- `script.js`

## How to use
1. Upload the files to a GitHub repository.
2. Enable **GitHub Pages**.
3. Open the Pages link.

## Demo flow
Splash → Terms → Digital ID → OTP → Feed → Map / Create / Friends / Profile → Humaid AI
