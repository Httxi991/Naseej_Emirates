# Naseej Emirates — iOS Competition Prototype (Elite)

This is a polished static prototype built for GitHub Pages.

## What it includes
- Mobile-first iPhone presentation
- Home → Terms → Digital ID → OTP → App flow
- Digital identity concept only (no manual age/name entry)
- Home feed with categories and access logic
- Explore map preview with directions
- Create event flow with category restrictions
- Friends list, invite flow, and chat
- Profile with settings and rewards
- Humaid AI as a conversation sheet
- Local demo state using `localStorage`

## Run locally
Open `index.html` in a browser.

## Deploy to GitHub Pages
Upload these files to a GitHub repository root, then enable GitHub Pages.
